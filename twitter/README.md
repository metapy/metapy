# Twitter Library

### Installation
	hg clone http://python-twitter.googlecode.com/hg/ python-twitter
	cp python-twitter/twitter.py .
	rm -rf python-twitter

	git clone https://github.com/simplegeo/python-oauth2.git
	cp -r python-oauth2/oauth2 .
	rm -rf python-oauth2
